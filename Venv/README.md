# Використання Excel Python
```
Бібілотеки залежать від серсії Python.
Якщо ми ставимо біблотеки, то вони встановлються
глобально при цьому у проекті ми можене не знати,
як вони правильно називаються і як їх поставити.
Установка пакетів (бібліотек) відбувається за допомогою
pip - менеджер пакетів, уміє їх ставити.

Ми завжди починамо із віртуальне середовище
має створити віртуальне середовище у нього ставимо пакети.
Віртуальне середовище потрібно активувати і тоді
можна у нього ставити пакети.
```

# Команди і робота з Excel - із cmd
```
py -m venv .venv
.venv\Scripts\activate.bat
```

# Встановлювання бібліотек
```
pip install openpyxl
python.exe -m pip install --upgrade pip
```

# Дуже важливо знати, які бібліотеки у Вас стоять
```
Цу команда зберігає у файл requirements.txt
список бібліотек, які у Вас у проекті
pip freeze > requirements.txt
```

# Якщо скачуємо проект з GitHub
```

бо це служова інформація
```
