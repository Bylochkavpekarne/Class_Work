print() - це функція, яка виводить текст на екран
int() - це функція, яка перетворює вхідне значення у число
str() - це функція, яка із вхідного значення робить рядок
a = True - зберігає булівське значення True ( істинна ) або False ( хиба )
\t - відступ табуляції
\n - перехід на новий рядок

перед рядком f - означає, що буде підстановка
f - рядок підстановки - буде підставлятися змінна {}

name = "Alice"
age = 30
print(f"Привет, {name}! Тебе {age} лет.")
# Вывод: Привет, Alice! Тебе 30 лет.

type() - перевіряє який тип зберігається в age
float() - перетворює у дробове число
lower() - приводить усі введені символи користувача у нижній регістр
